My Projects

Inventory Management system

The Inventory Management System API is a backend project designed to streamline the process of tracking and managing inventory for businesses. It is built using Golang and utilizes a PostgreSQL database for efficient data storage. The API allows users to perform CRUD (Create, Read, Update, Delete) operations on products, categories, and stock levels. Key features include: User Authentication: Secure login and registration using JWT tokens. Inventory Tracking: Real-time updates for product stock levels. Data Analytics: Generates reports on sales, restocking needs, and product trends. API Documentation: Comprehensive API documentation using Swagger for easy integration with frontend applications. This project demonstrates proficiency in API development, database management, and implementing secure, scalable backend systems.

Smart Home Control App

The Smart Home Control App is a mobile application developed to provide users with an intuitive interface for managing their smart home devices. Built using React Native for a cross-platform experience (iOS and Android), the app connects to a backend server implemented in Golang for handling device communication and user data. Key Features: Device Control: Users can turn on/off lights, adjust thermostats, and manage other connected devices from the app. User Authentication: Secure login with OAuth integration for personalized access. Real-Time Updates: Uses WebSocket connections for instant updates on device statuses. Voice Commands: Supports voice control through integration with virtual assistants like Google Assistant. Activity Logs: Tracks and displays a history of user actions for security and monitoring. This project showcases skills in full-stack mobile app development, backend API integration, real-time data handling, and user interface design.

Expense Tracker App

The Expense Tracker App is a mobile application designed to help users manage their personal finances effectively. Developed using Flutter, the app provides a seamless experience across both iOS and Android platforms. It connects to a Golang backend for secure data processing and storage, utilizing a cloud-based database (Firebase) for real-time syncing. Key Features: Budgeting: Users can set monthly budgets and track expenses by category (e.g., Food, Transport, Utilities). Expense Insights: Visual charts and analytics to help users understand their spending patterns. Data Synchronization: Real-time syncing of user data across multiple devices. Multi-Currency Support: Allows tracking of expenses in different currencies for international users. Secure Data: User data is protected with encryption and secure authentication (OAuth). This project highlights expertise in cross-platform app development, backend integration, real-time data handling, and financial analytics.
